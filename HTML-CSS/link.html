What are Links in HTML? A link in HTML is a clickable element (text or image) that redirects the user to another webpage, file, or resource. Links make a website interactive, user-friendly, and connected to other resources on the internet. Basic Syntax: Clickable Text Anchor Tag in HTML The (anchor) tag is used to create links. Basic Syntax: Link Text • href → Hyperlink Reference (destination URL). • Link Text → The clickable text visible to users. Example: Visit Example Website Here are the main types of links: a) Absolute URL (External Links) A complete web address including https:// or http://. Always points to the same resource, no matter where your website is hosted. Example : Go to Google b) Relative URL (Internal Links) A shorter path that points to files or pages inside your own website. Depends on your website’s folder structure. Example : About Us c) Email Links (mailto:) Opens the default email client to send a mail. Example : Send Email d) Phone Links (tel:) Useful for mobile users to dial numbers directly. Example : Call Us e) Internal Page Navigation (Bookmarks) Jump to a specific section on the same page. They are useful when the page is long, so users don’t need to scroll manually.
